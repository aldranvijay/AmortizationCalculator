To get the executable Microservice Jar,
Steps:
1. Open terminal into current directory
2. Run command "mvn install"
3. Go to /Source/target folder and copy AmortizationCalc-0.0.1-SNAPSHOT.jar

Note : Above jar file is executable and run under embedded server using command
       "java -jar AmortizationCalc-0.0.1-SNAPSHOT.jar"
